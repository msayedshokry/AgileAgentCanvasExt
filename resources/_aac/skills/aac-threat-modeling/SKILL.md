# Threat Modeling Skill

This skill provides AI-native 8-phase STRIDE threat modeling, security audits, risk assessments, and penetration test planning powered by the [fr33d3m0n/threat-modeling](https://github.com/fr33d3m0n/threat-modeling) repo (BSD-3-Clause, v3.2.0).

## Installation Required

This skill is available as a **Featured Skill Repo** and requires a one-time install:

1. Open the **Skill Catalogue** — click the ⊞ button on the canvas toolbar (or run `Agile Agent Canvas: Open Skill Catalogue`)
2. Go to the **Repositories** tab
3. Under **Featured**, find **threat-modeling** and click **Add Repo**
4. The extension clones the repo and imports the full skill automatically

> Requires git and `agileagentcanvas.userCataloguePath` to be configured in settings.

## What You Get After Installing

The full skill delivers:

- **8-phase STRIDE workflow** — Project Understanding → DFD Analysis → Trust Boundary Evaluation → Security Design Review → STRIDE Threat Analysis → Risk Validation → Mitigation Planning → Report Generation
- **Security knowledge base** — 1,900+ CWE/CAPEC/ATT&CK threat patterns, 107 security controls across 16 domains, OWASP ASVS/WSTG/MASTG compliance
- **Professional deliverables** — Risk Assessment Report, Risk Inventory, Mitigation Measures, Penetration Test Plan, DFD Diagrams, and per-phase audit trail
- **Attack path validation** — POC generation, attack chain tracing, multi-step exploit paths
- **Multi-language output** — en, zh, ja, ko, es, fr, de, pt, ru
